# money/ — everything money-related

Not just "investments" — budgets, student loans, rent, insurance,
income / expense logs, retirement planning, recovering from a scam …
any money topic goes here.

## What goes here

By the situation that matches you most right now:

| Your stage | Roughly what's here |
|------------|---------------------|
| Student | Monthly budget / student loans / scholarships / side income / big things you're saving for |
| Early career | Pay stubs / monthly spending / first savings / first insurance / credit-card debt situation |
| Mid-career | Portfolio table / mortgage / kid's education fund / multi-account coordination / retirement accounts |
| Senior / on the FIRE path | Asset allocation / FIRE math / SWR tests / multi-currency / cross-border tax |
| Retired / nearly retired | Cash flow / withdrawal strategy / long-term care insurance / supporting kids |
| Freelance / business | Income volatility management / taxes / emergency fund / company-vs-personal books |
| Financial stress | Debt list / cutting-back plan / negotiation strategy / help resources |

## What Muse can do with this

- **Budget / cutting back**: prioritize based on real spending (rent /
  food / subscriptions / commuting — what's easiest to trim)
- **Debt / loans**: repayment strategy (snowball vs avalanche vs minimum)
- **First insurance / first fund**: explain from zero what you need and
  don't need
- **Asset allocation**: current vs target ratios with specific rebalancing
- **Instrument due diligence**: tracked index / fee / size / liquidity /
  tax structure / historical drawdown
- **FIRE projection**: based on historical data + Monte Carlo for safe
  withdrawal rates
- **Risk scenarios**: 30% drawdown / job loss / serious illness — how
  long does your cash flow last
- **Major decisions**: cash-flow impact of buying a home / studying
  abroad / changing cities / having kids

## What Muse will NOT do

- No "will go up / down" predictions
- No recommending options / leveraged ETFs / meme stocks / leveraged-loan
  strategies
- Won't decide for you — lists options and the cost of each
- **For students / low-income stage**: Muse leads with saving, not with
  "invest your way out" — won't push retail-investing rhetoric at people
  who shouldn't be hearing it yet

## Notes

- Numbers can be redacted: write "around X thousand" or use encrypted
  relative values — Muse only needs magnitudes and ratios
- Don't write account passwords / API keys / card numbers into this
  archive — use a password manager
- The "why I made this call" matters more than "what I did" — record
  the reasoning at the time

## Getting started

1. Write one sentence about **your money situation right now** (e.g.
   "student, no income, lives off allowance" / "paycheck-to-paycheck,
   starting to save" / "mortgage 8 years to go, considering early
   payoff" / "60% to FIRE target")
2. List your main "accounts" (student card / paycheck account / brokerage
   / retirement account etc.)
3. Write **the one money problem you most want solved**

Muse will give tailored advice from this starting point.
