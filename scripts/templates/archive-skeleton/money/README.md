# money/ — 跟钱有关的一切

不是只有"投资"——预算、学贷、房租、保险、收支记录、退休规划、被骗追回……
钱的任何方面都放这里。

## 这里适合放什么

按你目前最相关的场景：

| 你的阶段 | 这里大概放什么 |
|---------|---------------|
| 学生 | 月度预算 / 学贷情况 / 奖学金 / 兼职收入 / 想存的大件 |
| 初入职场 | 工资条 / 月度开销 / 第一笔储蓄 / 第一份保险 / 是否要还信用卡 |
| 中产期 | 持仓表 / 房贷 / 子女教育金 / 多账户协调 / 退休账户 |
| 资深 / FIRE 路上 | 资产配置 / FIRE 计算 / SWR 测试 / 多币种 / 跨境税 |
| 退休 / 准退休 | 现金流 / 提取策略 / 长护险 / 子女资助计划 |
| 自由 / 经营 | 收入波动管理 / 报税 / 应急金 / 公司账与个人账 |
| 经济压力中 | 债务清单 / 节流计划 / 协商策略 / 求助路径 |

## Muse 能用这些做什么

- **预算 / 节流**：基于真实开销给优先级（房租 / 餐饮 / 订阅 / 通勤 哪里最容易省）
- **债务 / 学贷**：还款策略（雪球 vs 雪崩 vs 最低）
- **第一份保险 / 第一份基金**：从零讲清需要什么 / 不需要什么
- **资产配置**：当前比例 vs 目标，给具体调仓
- **标的尽调**：跟踪指数 / 费率 / 规模 / 流动性 / 税收结构 / 历史回撤
- **FIRE 推演**：基于历史数据 + Monte Carlo 给安全提取率
- **风险情景**：30% 回撤 / 失业 / 重病 下你的现金流够撑多久
- **大事决策**：买房 / 留学 / 换城市 / 生孩子 的现金流影响

## Muse 不会做什么

- 不预测"会涨 / 会跌"
- 不推荐期权 / 杠杆 ETF / meme 股 / 杠杆贷投资
- 不替你拍板，列出选项 + 各自代价
- **学生 / 低收入阶段**：Muse 会优先讲省钱，不引你去 "投资理财" 这种应试话术

## 注意事项

- 数字可以脱敏：写"约 X 万"或加密的相对值，Muse 用量级和比例就够
- 不要把账户密码 / API key / 银行卡号写在档案里（用密码管理器）
- 历史决策的"为啥这么做"比"做了啥"更重要——记录当时的判断依据

## 起步建议

1. 写一句**当前钱的处境**（如「学生没收入靠生活费」/「月光族刚开始想存钱」/
   「房贷还有 8 年，开始考虑提前还」/「FIRE 距离 60%」）
2. 列你的主要"账户"（学校卡 / 工资卡 / 余额宝 / 证券账户 / 退休账户 等）
3. 写一个**最想解决的钱的问题**

Muse 第一次进来会按你这个起点给针对性建议。
