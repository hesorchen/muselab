# work/ — 你在"做的事"

这个目录是"你大部分时间花在的事情"的产物。具体内容因人而异：

| 你的身份 | 这里大概放什么 |
|---------|---------------|
| 学生（本/硕/博/职校） | 课程笔记、论文、申研材料、毕设、奖学金申请 |
| 在职 | 简历、目标公司清单、项目复盘、绩效记录、面试笔记 |
| 自由职业 / 自雇 | 作品集、客户档案、合同、项目交付物 |
| 创业 | 商业计划、产品文档、团队 / 招聘、融资材料 |
| 全职照护 | 育儿 / 长辈照护日志、相关学习材料 |
| 退休 / Gap | 当前主要活动的记录（写作 / 志愿服务 / 学新东西） |

不必硬塞。空着或只放一两份当前最相关的就够。

## Muse 能用这些做什么

- **学生**：润色文书、申研定校建议、面试 / 套磁演练、论文反馈、课程难点讲解
- **在职**：简历改稿、目标公司情报、面试模拟、Offer 谈判（硬约束）、项目讲解话术
- **自由职业**：客户提案改稿、定价建议、作品集编排、报税思路
- **创业**：BP 反馈、用户访谈复盘、招聘 JD 撰写
- **照护 / 退休**：基于你的真实日常给具体建议，不假设你"该上班"

## 怎么放最有用

- 文档命名带日期 `YYYY-MM-xxx.md/pdf`
- 学习 / 工作笔记的"为什么这么做"比"做了什么"更重要——记录决策依据
- 涉及他人的内容（同事 / 客户 / 教授）脱敏：写"M" / "教授" / "PM" 代替名字

## 起步建议

只放一份**当前最相关**的：
- 学生 → 最近一份论文 / 最关心的课的笔记 / 在写的申请材料
- 在职 → 最新简历
- 自由职业 → 当前主推的 1-2 个作品 / 在跟的客户清单
- 其他 → 当前最大块时间花在的事情的笔记

Muse 第一次进来会主动问"想先聊哪件？"
