# health/ — 身体 / 健康相关

每个人都有一具身体——不分年龄、职业、阶段。这个目录放**关于你这具身体**
的一切。

## 这里适合放什么

- **体检报告**：每次的 PDF / 报告。命名 `YYYY-MM-体检-机构.pdf`
- **当前在用的药 / 补剂**：剂量、品牌、买入时间、为啥吃
- **运动 / 训练记录**：当前周计划 / 历史进度
- **睡眠 / 心率 / 步数**：可选导出（Apple Health / 华为运动 / 小米手环等）
- **就医记录**：处方、影像、医生建议、复诊计划
- **过敏 / 慢病**：长期跟着你走的健康信息
- **关心的人的健康**（可选）：父母 / 子女 / 伴侣的对应资料，建子目录分人放

## 不同人生阶段长什么样

| 你的情况 | 这里大概的样子 |
|---------|---------------|
| 学生 / 年轻 | 入学体检 / 视力 / 体能 / 偶尔运动伤记录 |
| 备孕 / 怀孕 / 产后 | 产检报告 / 营养记录 / 产后恢复 |
| 中年 | 年度体检 / 长期指标趋势 / 慢病管理 / 运动调整 |
| 三明治一代 | 自己 + 父母双线（父母放 `parents/` 子目录） |
| 老年 / 准退休 | 慢病管理 / 药物表 / 跌倒预防 / 老伴健康 |
| 慢病 / 康复期 | 复诊计划 / 药物表 / 症状日记 |
| 心理健康关注 | 咨询笔记 / 睡眠日记 / 情绪记录（可加密单放） |

## Muse 能用这些做什么

- 跨年体检对比：解读哪些指标在变好 / 变坏 / 接近临界
- 补剂方案审视：剂量是否过量 / 与药物相互作用 / 是否有更有效替代
- 训练计划调整：根据近期数据 + 你的目标重新排
- 体检前预问：提醒该加测哪些项目
- 出现异常指标时：给循证下一步（找哪科医生 / 该查什么）
- 父母 / 长辈健康追踪：进入哪个阶段该筛查什么

## 注意事项

- 数字一定要**原样保留**——Muse 会引用具体值给建议
- 单位写清（mmol/L vs mg/dL；ng/mL vs nmol/L 差很多）
- 病案号 / 身份证号请脱敏后再放
- 心理健康 / 性健康 / 精神类用药相关：考虑单独加密文件，看你自己舒适度

## Muse 不会做什么（重要约束）

- 不下诊断
- 不开处方
- 不夸大补剂效果（与药物边界清晰）
- 不推 MLM 保健品（无限极 / 安利 等，除非有独立证据）
- 任何重要决定都会反复提醒**看医生**

## 起步建议

1. 把最近一次体检报告放进来（学生没体检的话写一句「身高 X、体重 Y、目前
   没大问题」也够）
2. 列「当前在吃的所有药 / 补剂」
3. 写一行**当前主要健康关注**（或「没有特别关注」）

Muse 第一次进来会主动问"要先解读体检报告吗 / 要不要聊运动？"
