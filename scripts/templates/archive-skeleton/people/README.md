# people/ — 你关心的人

不只是"家庭"。可以是伴侣、父母、子女、室友、铁哥们、长期朋友、导师、
学生、网友、宠物……谁对你重要、谁现在需要你多花心思，就在这里有一席之地。

## 这里适合放什么

- **关键关系档案**：每个重要的人一份简短笔记（年龄 / 健康 / 阶段 / 你跟他/她
  的关系动态）
- **家庭决策**：照护安排、教育方案、家务分工
- **伴侣 / 婚姻**：共同目标、近期重要事、关系维护笔记
- **父母 / 长辈**：健康清单、保单、子女联络、重要日子
- **子女**（如有）：成长记录、学校事、健康
- **朋友 / 圈子**：长期重要朋友的近况、你们的共同事
- **保险 / 法律**：与人相关的保单、信托、遗嘱
- **重要日期**：生日 / 纪念日 / 重大手术日 / 移居日

## 不同人生阶段长什么样

| 你的情况 | 这里大概的样子 |
|---------|---------------|
| 学生 / 独居 | 父母简况 + 几个铁哥们 + 室友（如有） |
| 异地恋 / 异地婚 | 伴侣简档 + 维护节奏 + 共同重要日子 |
| 已婚有孩子 | 伴侣 + 孩子 + 双方父母 + 育儿分工 |
| 三明治一代 | 父母（照护） + 孩子（教育） + 自己（夹中间）—— Muse 会跨段建议 |
| 单身丁克 | 父母 + 自选家人 + 长期朋友圈 |
| 退休 / 老年 | 老伴 + 子女 + 老友 + 医疗联系人 |
| 离异 / 重组家庭 | 复杂关系图 → 用一份「关系地图」md 文件 |

## Muse 能用这些做什么

- **跨领域决策**：把家人信息 + 你的财务 + 你的时间 关联给建议（如「父母手术 +
  你今年现金流 + 你的休假天数」一起评估）
- **照护安排**：提示父母 / 长辈进入哪个阶段该筛查什么、该备什么
- **保险审视**：覆盖是否有缺口 / 重复 / 过期
- **关系维护**：重要日子提醒 + 个性化建议（结合你跟对方的近况）
- **大事决策**（生育 / 换房 / 父母搬家 / 异地团聚）：列涉及面、前置条件、推翻条件

## 注意事项

- **这是最敏感的目录**。强烈建议给整个 muselab archive 加全盘加密，**不要**
  sync 到公共云
- 涉及他人的信息：写「M」「父亲」「室友 A」代替真名也行，关键是 Muse 知道
  关系
- 健康信息按人分文件夹放，命名清楚
- 不要写社保号 / 身份证号 / 银行账号——这些放密码管理器
- 离异 / 法律纠纷相关材料：考虑单独加密文件，不裸放

## 起步建议

1. 写一段「当前我最关心的人 / 关系是谁，他们怎么样」
2. 列 3-5 个对你最重要的人（一句话各自情况）
3. 写一个**当前关于这些人最想解决的问题**

Muse 第一次进来会按这个起点提建议。
