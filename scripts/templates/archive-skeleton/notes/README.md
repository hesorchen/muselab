# notes/ — 杂项笔记

## 该放什么

- **学习笔记**：读书 / 课程 / 论文摘要
- **灵感 / 想法**：随手记，分主题或按日期都行
- **生活记录**：周记 / 月复盘 / 年度总结
- **不归属其他目录的所有东西**

## Muse 能用这些做什么

- 全文检索 + 关联推荐：你聊到 X，Muse 自动告诉你"你两个月前在某篇笔记里写过类似的"
- 周 / 月 / 年复盘：基于你的笔记帮你抽出主题
- 把零散想法整理成可执行任务

## 起步建议

按"我现在最感兴趣 / 最常思考"的 3-5 个主题各建一个子目录或一个 md 文件
开始记。不用一开始就完美分类，三个月后自然会知道该怎么重组。
