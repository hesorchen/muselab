CORE IDEA

你的私人档案，
就是 AI 的工作集。

看着 Muse 想清楚
再动手。

DIFF STRIPS

改文件就出
红绿 diff，看完再决定。

Muse 每改一个文件，气泡里直接渲染 inline diff——绿加红减、带 +N −M 计数。一眼看出动了哪里，不用切到 git。点击 head 折叠，长对话不挡视线。

所有 Edit / Write / MultiEdit 工具调用都附 diff · 当前 turn 自动展开，历史折叠

MOBILE PWA

同一台服务器，
手机也能无缝接着用。

添加到主屏，开起来就是全屏 app（standalone display，没有地址栏）。底部三按钮 tab bar：Files / Preview / Chat 单页切换。下拉刷新文件树，长任务跑完 Web Push 推送提醒。归档自始至终在你自己的主机上，不过云。

PWA manifest · 主题色 #0f1115 · maskable 图标 · Apple Touch icon · 无需上架
